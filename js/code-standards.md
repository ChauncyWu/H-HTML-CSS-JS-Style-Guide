# Code Standards

### 1. 文档注释

```javascript
/*
 * @Author: Chauncywu
 * @Date: 2020-09-27 10::35
 * @LastEditTime: 2021-01-27 11:20:15
 * @LastEditors: Please set LastEditors
 * @Description: Cookie相关方法
 * @FilePath: 文件路径
 */
```

### 2. 函数注释

```javascript
/**
 * @description: 函数描述
 * @param {number} num 参数1的说明
 * @return {number} 返回值的说明
 * @example
 *
 *
 */
function test(num) {
  var n += 1
  return n;
}
```

| 符号 | 用法 |
| :--- | :--- |
| @param | {类型} 参数名 描述 |
| @return | {类型} 描述 |
| @author | 作者 |
| @version | 版本号 |
| @todo | 待办 |
| @api | 提供给第三方使用的接口 |
| @property-read | 只读属性 |
| @property-write | 只写属性 |
| @const | 常量 |
| @example | 示例 |
| @copyright | 版权声明 |
