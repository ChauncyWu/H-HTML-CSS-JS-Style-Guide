# General

### 1. 缩进使用soft tabs（4个空格）

{% hint style="info" %}
为什么使用soft tabs而不是hard tabs？

1. 在任何平台上查看一致的代码；
2. 使代码易于复制和粘贴以供在线讨论和共享，因为大多数在线观看者将Tab字符渲染为4个或更多空格；
{% endhint %}

### 2. 文件名命名使用小写字母，使用（\_）下划线分隔，带连词线（-）的单词不变

{% hint style="info" %}
为什么文件名要小写？

1. 可移植性：保证跨平台的稳定性
2. 易读性：小写比大写易读，驼峰不适合全部是大写的
3. 易用性：易于与系统文件区分
4. 便捷性：减少使用shift键，保护手部

为什么使用（\_）下划线分隔？

1. 方便与带连词线的单词区分
2. 方便全选文件名
{% endhint %}
